# Demo Stand Instructions

This workspace now contains a full **Theme Demo** to showcase the power of CSS decoupling to your client.

## Files
- `index_demo.html`: **Main Demo File**. Open this in the browser. It contains the Theme Switcher widget (bottom right).
- `assets/css/style-eco.css`: New "Eco/Organic" theme.
- `assets/css/style-neon.css`: New "Cyberpunk" theme.
- `assets/css/style-luxury.css`: Your Luxury theme.
- `assets/css/style-newyear.css`: Your New Year theme.

## How to use
1. Open `index_demo.html` in a browser.
2. Click the **Magic Wand / Gear icon** in the bottom right corner.
3. Select a theme from the menu to instantly switch the design without reloading the page.
4. **New Year Theme**: Notice that selecting this also activates the Snow and Garland effects (JavaScript injected dynamically).
5. **Other Themes**: Observe how fonts, colors, roundings, and shadows change completely while the HTML structure remains identical.

## Themes Included
1. **Base**: The original design.
2. **Luxury Gold**: Dark, premium, expensive feel.
3. **New Year**: Festive, red/green, snow animations.
4. **Cyberpunk**: Neon, high contrast, tech-focused.
5. **Eco Organic**: Green, beige, soft, sustainable feel.

## Technical Value for Client
- **A/B Testing**: We can test which design converts better without rewriting code.
- **Seasonal Marketing**: Switch to "Black Friday" or "Christmas" mode in seconds.
- **SEO Safety**: Google sees the same consistent HTML structure, preserving your rankings.
